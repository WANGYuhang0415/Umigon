1.Simple

  • Send your data, receive back some useful information.
  • Standard GET or POST requests with your data and your API key, returning Json objects - that's it!

2.Fast

  • We love speed: our code is optimized to take advantage of multicore servers.
  • You can Gzip your data to reduce bandwith and shorten even more the request time - just add a header "Content-Encoding: gzip" to your requests.

3. Transparent

  • Clean API doc provided for php, java, ruby, and node.js.
  • The scientific research behind each service is made available openly.
